Detailed Description

template<typename T, index_t N>
class geom::Rect< T, N >

An N-dimensional axis-aligned box.

This class works naturally with both 1D and multi-dimensional ranges. 1D ranges have point type T, while multi-dimensional ranges have point type Vec<T,N>. A typedef of this may be accessed via: 

Rect<T,N>::point_t

Example:

  • Rect<double,4>::point_t is Vec<double,4>; getMin() returns a Vec4d.
  • Rect<double,1>::point_t is double; getMin() returns a double.

Rects obey the standard interval conventions, in that lower extremes are inclusive and upper extemes are exclusive.
