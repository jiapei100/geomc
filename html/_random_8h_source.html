1 /*
2  * Random.h
3  *
4  * Abstract class for random number generation.
5  * Implementations provide random bits by overriding rand32(), which should return
6  * no fewer than 32 random bits.
7  *
8  * Client-friendly random numbers are exposed via Random::rand<T>(). Specializations
9  * are provided for a selection of primitive T. Specializations for other T may be written,
10  * without the need to create new subclasses.
11  *
12  *
13  * The general contract of rand() which should be followed by all specializations:
14  *
15  * rand(void)
16  * - if T is discrete, than this function should return values evenly over the entire
17  * space of possible T.
18  * - if T is a continuous type, then this function should return values evenly
19  * over an analogue of the interval [0,1], inclusive.
20  *
21  * rand(T max)
22  * - should return random values evenly distributed over all possible states between
23  * 0 (or the T analogue of 0) and the value of <max>
24  *
25  * rand(T lo, T hi)
26  * - should return random values evenly distributed over all possible states
27  * bounded by <lo> and <hi>
28  *
29  *
30  * (!) This class was designed to be portable between systems with different integer sizes,
31  * although this feature has not been tested (!)
32  *
33  * Created on: Feb 22, 2009
34  * Heavily refactored in April 2011
35  * Author: Tim Babb
36  */
