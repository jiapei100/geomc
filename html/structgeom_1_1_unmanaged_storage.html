Detailed Description

template<typename T, index_t N>
struct geom::UnmanagedStorage< T, N >

ArrayStorage with templated static or dynamic size, and without reference counting (allocated data is deleted upon destruction).

Template Parameters
TElement type.
NSize of the array, or 0 for dynamic size.

Preferred for when ownership never changes hands and/or memory management is not needed.
