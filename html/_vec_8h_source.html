202 /* Declare the Vec type that we'll actually use, as a subclass of the hidden base template class.
203  * We do this so that all sizes of vector can share common code, since a template that is merely specialized
204  * does not implicitly inherit functionality from the base template. With this method, Vec<T,2> can be a specialization
205  * of Vec<T,N>, but still keep the functionality of VecBase<T,N>.
206  *
207  * All vectors are sizeof(T)*N and can be packed.
208  */
